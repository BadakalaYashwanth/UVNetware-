# UV Netware AI Analytics Dashboard

This is the frontend dashboard for visualizing marketing insights generated by the AI analytics pipeline.

## Features
- **Real-Time Sync**: Automatically refreshes every 10 seconds to reflect backend changes.
- **Modern UI**: Glassmorphic dark mode design with responsive layouts.
- **Interactive Charts**: Powered by Chart.js (ROI, Devices, Funnel, Geo).
- **Intelligent Insights**: Displays anomalies and budget recommendations directly from `output_insights.json`.

## How to Run

### 1. Start a Local Server
Due to browser security (CORS), you must run a simple server from the **root** of the project:

```bash
# Using Python
python -m http.server 8000

# or Using Node.js (npx)
npx serve .
```

### 2. Open the Dashboard
Navigate to:
[http://localhost:8000/frontend/](http://localhost:8000/frontend/)

## Data Flow
`Backend (main.py)` &rarr; `output_insights.json` &rarr; `Frontend Dashboard`
